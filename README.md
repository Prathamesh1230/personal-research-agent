# Personal Research Agent

An AI-powered research agent that autonomously searches the web, synthesizes information from multiple sources, and generates a structured research report with a downloadable PDF.

## Features
- Autonomous web search using Tavily API
- AI-powered report generation using Google Gemini
- Downloadable PDF reports
- Clean Streamlit web interface
- Quick Summary or Detailed Report modes

## Tech Stack
- Python
- LangChain + LangGraph
- Google Gemini API
- Tavily Search API
- Streamlit
- ReportLab

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your API keys to `.env`
4. Run: `streamlit run app.py`
